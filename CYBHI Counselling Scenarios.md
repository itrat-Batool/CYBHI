# **Counseling Scenarios**

## **1\. Counseling Session with Full Class (Chargeable per Student)**

In this scenario, counselors can schedule a planned or ad‑hoc, non‑incident counseling session with a full class. This session is billed individually for each student who attends.

### **Workflow**

- Students check in by scanning a **barcode**, which prompts them to **enter their student number**. Names may also be entered if needed, and any names included in counselor notes are automatically **redacted**.  
    
- The system does **not** record the full session. Instead, it tracks **total class duration**, which can be auto‑tracked or **manually updated** by the instructor.  
    
- After the session, the counselor provides a **written summary** of what was discussed or taught.  
    
- The system extracts the appropriate **billing code** based on the counselor’s summary.  
    
- Attendance is logged for billing, but identifiable information remains protected.  
    

### **Key Features**

- Barcode check‑in with student number and optional name entry.  
    
- Automatic or manual duration tracking.  
    
- Post‑session summary instead of full recording.  
    
- Billing code auto‑extracted from counselor input.  
    
- Redaction of any student names in notes.  
    

## **2\. Incident-Triggered Individual Counseling Sessions**

These sessions occur after an incident involving one or more students. This workflow is currently being worked on.

### **Workflow**

- An incident is reported and linked to the involved students, with redaction applied where needed.  
    
- Counselors conduct **individual counseling sessions**, each stored separately.  
    
- For now, counselors can simply **record what was discussed** in the session. No detailed or live session capture is required.  
    
- Each counseling session is **billable**, with billing codes identified using the same approach currently in place.  
    

### **Key Features**

- Simple, note‑based documentation per counseling session.  
    
- Billable per individual session.  
    
- Additional enhancements under development.  
    

### **Summary**

This document outlines two primary counseling session flows:

- **Class-based charged sessions** with barcode check‑in, duration tracking, summaries, and redaction.  
    
- **Incident-driven individual sessions**, currently being improved, focused on counselor‑recorded discussions and billing.  
    

Both flows ensure privacy, compliance, accurate billing, and structured documentation.
